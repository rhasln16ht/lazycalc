# LazyCalc
A minimal, fast, and easy-to-use command line calculator built with modern C++.

## Features 
- Interactive CLI  mode.
- Basic arithmetic operations.
- Minimal and fast.
- Only for UNIX-like systems (such as GNU/Linux) and Android via [Termux](https://f-droid.org/id/packages/com.termux/).

##  Build & Run
```bash
make
./build/lazycalc
```
