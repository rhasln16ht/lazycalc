# LazyCalc
A simple and lightweight terminal calculator written in C++.

## Features (v0.1.0)
- Interactive CLI  mode.
- Basic arithmetic operations.
- Minimal and fast.

## Changelogs
## [0.1.0] - 2026-02-26

### Added

### Deprecated

- "Q" as operator to exit, will be removed in v0.2.0
